Specifications

The cryoboard offers a different way of computing in terms of the application interface provided to the user through the keyboard. For example applications like the calculator, photoshop, etc. are customized for the usage of the user.Given the design the bluetooth connectivity is another feature that makes it unique.


WIFI Connectivity

WIFI connectivity enabling wireless connection to the internet for the the cryoboard using 802.11 B/G/N WIFI adapter.

Raspberry pi 3

The brains behind the processing work of the cryoboard is the Raspberry pi 3 with a 1GB RAM and a 40 pin header.

Basic Parameters

15.6 inch display full HD 1080p IPS LCD, touchscreen.



DESIGN

With a sleek,unique and cuztomizable design the cryoboard feels wonderful to work with; not forgetting the touch capability which makes the functioning of the cryoboard seamlessly smooth.
